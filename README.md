# 🛠️ AI-Assisted Unit Converter

## 🎯 Objective
This project is a simple, functional Unit Converter built with HTML, CSS, and JavaScript. It was developed as part of the "AI-Assisted Development and Git Mastery" assignment to demonstrate proficiency in Git CLI version control and collaborative coding with Generative AI.

## 🚀 Live Demo
You can view and interact with the deployed application here:
👉 https://sandtrolilol14.github.io/AI-Unit-Converter/

## 🤖 AI Collaboration
This project was developed using a Pair-Programming approach with an AI assistant. The AI was utilized to:
* Draft the initial HTML structure and CSS styling.
* Write the JavaScript logic for unit conversions (e.g., Kilometers to Miles, Celsius to Fahrenheit).
* Debug and format the code to ensure it is clean, readable, and properly commented.

## 💻 Git CLI Workflow Demonstrated
Throughout the development of this project, the following Git concepts were applied via the integrated terminal:
* Repository Initialization (`git init`).
* Staging and Tracking files (`git add`, `git status`).
* Committing changes with descriptive messages (`git commit`).
* Branch management for new features (`git branch`, `git checkout`).
* Merging feature branches back into the main workflow (`git merge`).
* Connecting and pushing to a remote GitHub repository (`git push`).
